# 智能待办 Smart-Tasks

> 一款专为 Android 设计的智能任务管理工具，集任务管理、番茄钟、AI 助手于一体。

## 📦 下载安装

👉 **[下载最新版 APK](https://github.com/wolf28014/work/releases/latest)**

- 支持系统：Android 7.0 (API 24) 及以上
- 已签名，可直接安装

### 安装步骤

1. 点击上方链接下载 APK 文件
2. 在文件管理器中找到下载的 APK，点击安装
3. 系统提示"未知来源"时，按引导允许安装即可
4. 桌面会出现 **智能待办** 图标（翡翠绿对勾 ✅）

> 💡 也可以在 App 内「设置 → 检查更新」直接下载新版本

## ✨ 核心功能

### 5 个视图

| 视图 | 说明 |
|------|------|
| 📋 **任务** | TF-IDF 搜索、按标签分组、4 种筛选、4 种排序、批量删除 |
| 📊 **看板** | 左右布局，分类侧边栏 + 任务列表，拖拽切换状态，批量删除 |
| 📅 **日历** | 月历视图，按状态颜色显示任务圆点，点击日期查看当天任务 |
| 🍅 **番茄钟** | 25 分钟专注 + 5 分钟休息，关联任务、震动提醒、系统通知 |
| 📈 **统计** | 完成率、状态分布、优先级分布、近 7 天趋势、番茄钟统计、AI 周报 |

### 任务模型

- 标题、描述、截止日期、优先级、状态机
- 重复任务（每天 / 每周 / 每月）
- 标签（8 种颜色，可管理）
- 子任务（可勾选，支持展开/收起）
- 番茄钟计数
- 软删除 + 30 天回收站

### AI 助手（6 大场景）

| 位置 | 功能 |
|------|------|
| 顶部 ✨ AI 按钮 | AI 对话助手 |
| 任务编辑器标题 | AI 智能解析（自然语言提取任务信息） |
| 任务编辑器子任务 | AI 拆解子任务 |
| 任务编辑器底部 | AI 任务总结 |
| 番茄钟页面 | AI 专注建议 |
| 统计页面 | AI 周报 |

> AI 功能需在「设置 → AI」中配置 API Key，支持智谱 GLM（免费）、OpenAI、DeepSeek、Moonshot 等

### 账号与云同步

- 邮箱 + 密码注册登录
- 手机号 + 短信验证码登录
- 任务、番茄钟、标签自动同步到云端
- 多设备数据共享
- 数据安全（Supabase RLS 行级安全）

### 其他特性

- 🎨 8 个高级预设背景 + 自定义图片上传
- 🌗 深色模式
- 💾 数据导入导出（JSON / CSV）
- 🗑️ 回收站（30 天自动清理）
- 🏷️ 标签管理（添加、删除、修改颜色）
- 📱 全面屏手势（系统返回键原生支持）
- 🔔 番茄钟提醒（震动 + 系统通知）
- ⭐ Pro 会员（兑换码激活）

## 🔧 AI 配置

在 App 内「设置 → AI」中填入：

| 平台 | Base URL | 模型 |
|------|---------|------|
| 智谱 GLM（推荐，免费） | `https://open.bigmodel.cn/api/paas/v4` | `glm-4-flash` |
| OpenAI | `https://api.openai.com/v1` | `gpt-4o-mini` |
| DeepSeek | `https://api.deepseek.com/v1` | `deepseek-chat` |
| Moonshot | `https://api.moonshot.cn/v1` | `moonshot-v1-8k` |

不配置 AI 也能正常使用所有核心功能。

## 🔒 权限说明

| 权限 | 用途 |
|------|------|
| INTERNET | AI 功能联网、账号、云同步 |
| VIBRATE | 番茄钟结束震动提醒 |
| WAKE_LOCK | 番茄钟运行时保持屏幕 |
| POST_NOTIFICATIONS | 番茄钟完成通知（Android 13+） |

无后台数据上传、无账号追踪、无广告。
